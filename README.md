### HR360

#### Project Overview

Hr360 is a Fullstack dashboard that tracks and manages employee activities,payroll etc

#### Features:

```bash

Clean, responsive UI
Complete Authentication Flow
Progressive web app
Backend API

```

#### Technologies

```bash
React
NextJS
Tailwind
Typescript
```
